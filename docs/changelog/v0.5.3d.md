## Overview

This version creates a comprehensive profile management service for shell profiles.

## Changes

### Added

- **Event Models:**
  - `ProfileChangeType` enum (Added, Updated, Deleted, DefaultChanged, Reset)
  - `ProfilesChangedEventArgs` class

- **`IShellProfileService`** interface with:
  - Read: GetAllProfilesAsync, GetVisibleProfilesAsync, GetProfileAsync, GetDefaultProfileAsync
  - Write: CreateProfileAsync, UpdateProfileAsync, DeleteProfileAsync, SetDefaultProfileAsync, DuplicateProfileAsync
  - Bulk: ResetToDefaultsAsync, ImportProfilesAsync, ExportProfilesAsync
  - Utility: GetEffectiveSettings

- **`ShellProfileService`** implementation with:
  - JSON persistence to app data directory
  - Thread-safe operations using SemaphoreSlim
  - Automatic built-in profile generation from detected shells
  - Default profile resolution with 3-level fallback
  - Shell path validation and auto-detection
  - Import/export with new ID generation
  - ProfilesChanged event notifications
