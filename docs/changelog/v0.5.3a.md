# v0.5.3a – Shell Detection Service Enhancement

> Extended shell detection API with public methods for type detection, validation, PATH search, and version extraction.

## Overview

This version extends the v0.5.1e shell detection implementation by exposing internal helper methods as public interface methods and adding new shell type support.

## Changes

### Added

- **`IShellDetectionService.DetectShellType(string)`** – Detects shell type from executable path
- **`IShellDetectionService.ValidateShellPath(string)`** – Synchronous path validation
- **`IShellDetectionService.FindInPath(string)`** – PATH environment search utility
- **`IShellDetectionService.GetShellVersionAsync(string, CancellationToken)`** – Version extraction

### Enhanced

- **`ShellType` enum** – Added `Tcsh`, `Ksh`, `Wsl` values
- **`ShellInfo` record** – Added `IconPath` and `Description` properties

### Fixed

- WSL shells now correctly report as `ShellType.Wsl` instead of `ShellType.Bash`
