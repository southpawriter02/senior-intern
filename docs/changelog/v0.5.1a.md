# v0.5.1a - Project Setup & Dependencies

**Date**: 2026-01-17
**Status**: IMPLEMENTED

Terminal infrastructure foundation. Adds cross-platform PTY packages and creates folder structure for terminal subsystem.

## Added

### NuGet Packages

- `Pty.Net` 0.1.16-pre - Cross-platform pseudo-terminal library (WinPTY on Windows, native PTY on Unix)
- `System.IO.Pipelines` 8.0.0 - High-performance asynchronous stream handling

### Directory Structure

- `src/AIntern.Core/Models/Terminal/` - Terminal model classes (v0.5.1b)
- `src/AIntern.Core/Terminal/` - ANSI parser logic (v0.5.1c)
- `src/AIntern.Services/Terminal/` - Terminal service implementations (v0.5.1d-e)

## Modified

### Package Configuration

- `Directory.Packages.props` - Added central package versions for terminal support
- `src/AIntern.Services/AIntern.Services.csproj` - Added package references

## Technical Details

### Pty.Net Capabilities

| Platform | Implementation |
|----------|---------------|
| Windows | WinPTY / ConPTY (Win10 1809+) |
| macOS | Native `/dev/pty*` |
| Linux | Native `/dev/pty*` |

### Future Usage (v0.5.1d)

```csharp
// Pty.Net API overview
var pty = await PtyProvider.SpawnAsync(new PtyOptions
{
    Name = "Terminal",
    Cols = 80,
    Rows = 24,
    Cwd = workingDirectory,
    App = shellPath
});

// Read output
await pty.ReaderStream.ReadAsync(buffer);

// Write input
await pty.WriterStream.WriteAsync(input);

// Resize terminal
pty.Resize(cols, rows);
```
