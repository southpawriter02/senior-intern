## Features

### Layout (4 rows)
- Row 0: Header with workspace name, New File/Folder, Refresh buttons
- Row 1: Filter box with search icon and clear button
- Row 2: Loading progress bar (when IsLoading)
- Row 3: TreeView, empty state, or error message

### TreeDataTemplate
- Chevron icons for folders (expand/collapse)
- File icons via FileIconConverter
- Inline rename TextBox
- Modified indicator (unsaved changes ellipse)

### Context Menu
- Open, Attach to Chat (files only)
- New File, New Folder
- Rename, Delete
- Copy Path, Copy Relative Path, Reveal in Finder

### Keyboard Shortcuts
- Enter: Open file / Toggle folder
- F2: Begin rename
- Delete: Delete item
- Right/Left: Expand/Collapse folder
- Ctrl+C: Copy path
- Ctrl+Shift+C: Copy relative path
- Ctrl+N: New file
- Ctrl+Shift+N: New folder
- Ctrl+R: Refresh

### Code-Behind Handlers
- OnTreeViewDoubleTapped
- OnTreeViewKeyDown
- OnRenameKeyDown
- OnRenameLostFocus (auto-commit)
