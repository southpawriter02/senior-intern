## Features

- **Live Countdown**: Real-time timer showing time remaining
- **Auto-Hide**: 10 second default visibility
- **Expiring Soon**: Pulse animation when < 30 seconds
- **Undo Button**: One-click undo with loading state
- **Dismiss Button**: Close immediately
- **UndoManager Integration**: Event-driven state updates
- **Entrance Animation**: Slide-up with fade-in
