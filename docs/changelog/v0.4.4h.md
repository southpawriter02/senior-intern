# v0.4.4h Changelog - Chat Integration

**Released**: 2026-01-16

## Summary

Bridge layer connecting the multi-file proposal subsystem (v0.4.4a-g) with the chat UI.

## Features

### Core Interface (`src/AIntern.Core/Interfaces`)
- **`IProposalDetectionService`** - Contract for proposal detection in messages

### Services (`src/AIntern.Services`)
- **`ProposalDetectionService`** - Detects multi-file proposals with caching
- **`ProposalDetectionOptions`** - Configurable thresholds and ignored languages

### Desktop Services (`src/AIntern.Desktop/Services`)
- **`ChatProposalCoordinator`** - Orchestrates Preview/Apply/Undo workflows
- **`ProgressReporterAdapter`** - Adapts ViewModel to IProgress interface

### Models (`src/AIntern.Desktop/Models`)
- **`UndoToastEventArgs`** - Event args for undo toast display
- **`BatchUndoCompletedEventArgs`** - Event args for batch undo completion

## Tests

- 8 new tests for `ProposalDetectionService`

## Dependencies

- v0.4.4a-g: All multi-file creation components
- v0.4.1: Existing chat infrastructure
