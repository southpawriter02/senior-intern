# v0.3.4g: File Explorer Integration

**Date:** 2026-01-15

## Summary

Implements file attachment from the File Explorer to the chat context via drag-drop and context menu.

## Changes

### ChatView.axaml
- Added `DragDrop.AllowDrop="True"` property
- Added drop zone indicator with visual feedback (blue overlay with icon)

### ChatView.axaml.cs
- `InitializeDragDrop()` - Attaches drag-drop event handlers
- `OnDragEnter` - Shows drop zone indicator on file drag
- `OnDragOver` - Sets drag effects
- `OnDragLeave` - Hides drop zone indicator
- `OnDrop` - Attaches dropped files via MainWindowViewModel

### MainWindowViewModel.cs
- `AttachFileAsync(string filePath)` - Reads file, estimates tokens, logs attachment
- Updated `OnFileAttachRequested` to call `AttachFileAsync`

## Existing Features Used
- `FileExplorerViewModel.AttachToContext` command (already implemented)
- `FileExplorerViewModel.FileAttachRequested` event (already implemented)
