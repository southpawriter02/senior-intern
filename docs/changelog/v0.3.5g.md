# v0.3.5g: Keyboard Shortcuts Manager

**Date:** 2026-01-15

## Summary

Centralized keyboard shortcuts manager for handling global and context-specific shortcuts.

## New Files

### ShortcutInfo.cs
- Key, Modifiers, CommandId, Description, Category properties
- `DisplayGesture` computed property (e.g., "Ctrl+S")

### KeyboardShortcutService.cs
- `IKeyboardShortcutService` interface
- `Register(key, modifiers, commandId, description)` - add shortcuts
- `HandleKeyPress(KeyEventArgs)` - match and fire events
- `GetAllShortcuts()` - returns sorted list by category
- 18 default shortcuts across 7 categories

## Modified Files

### MainWindow.axaml.cs
- `OnOpened`: Wires up shortcut service from ViewModel
- `OnShortcutCommandRequested`: Routes commands to ViewModel
- `OnKeyDown`: Calls `HandleKeyPress` first

### MainWindowViewModel.cs
- Added `KeyboardShortcutService` property
- Added `ExecuteCommandAsync(commandId)` for command routing

### ServiceCollectionExtensions.cs
- Registered `IKeyboardShortcutService` as singleton
