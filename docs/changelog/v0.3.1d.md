## Features

### Directory Operations
- `GetDirectoryContentsAsync` - List with sorting (folders first)
- `CreateDirectoryAsync`, `DeleteDirectoryAsync` (recursive)

### File Operations
- `ReadFileAsync`, `WriteFileAsync`, `DeleteFileAsync`
- `RenameAsync`, `CopyFileAsync`, `MoveAsync`

### File Watching
- 200ms debounce window
- Last event wins for rapid changes to same path
- `FileSystemChangeEvent` with Path, OldPath, ChangeType

### Binary Detection
- Signature matching: PNG, JPEG, GIF, PDF, ZIP, EXE, RAR
- Null byte detection for unknown extensions

### .gitignore Support
- Pattern loading from workspace
- Negation patterns (`!important.log`)
- Last-match-wins semantics
