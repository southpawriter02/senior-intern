## Features

### Workspace Commands
- `OpenWorkspaceCommand` - Folder picker
- `CloseWorkspaceCommand` - Reset state
- `RefreshCommand` - Reload tree

### File/Folder Commands
- `NewFileCommand` / `NewFolderCommand` - With inline rename
- `RenameCommand` / `DeleteCommand`
- `OpenFileCommand` - Raises FileOpenRequested

### Path Commands
- `CopyPathCommand` / `CopyRelativePathCommand`
- `RevealInFinderCommand` - Cross-platform (Mac/Win/Linux)

### Filter System
- 200ms debounce timer
- Recursive filter application
- Auto-expand matching parents

### Events
- `FileOpenRequested` - Open file in editor
- `FileAttachRequested` - Add to chat context
- `DeleteConfirmationRequested` - Confirm delete

### IFileTreeItemParent Implementation
- `LoadChildrenForItemAsync` - Lazy load children
- `RenameItemAsync` - Handle rename from child
- `OnItemExpansionChanged` - Persist expanded state
- `ShowError` / `GetRelativePath`
