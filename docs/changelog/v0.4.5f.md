# v0.4.5f Changelog - Keyboard Shortcuts

**Released**: 2026-01-16

## Summary

Enhanced keyboard shortcut system with context-aware dispatch, customization support, and persistence.

## Features

### Core Models (`src/AIntern.Desktop/Models`)
- **`KeyboardShortcut`** - Immutable struct with:
  - Key + modifiers combination
  - Parsing from string notation (`Ctrl+Shift+A`)
  - Platform-aware display strings (⌘⇧A on macOS)
  - Equality and hash code implementation
- **`ShortcutContext`** - 10 UI contexts (Global, ChatInput, CodeBlock, DiffViewer, etc.)
- **`ShortcutCategory`** - Settings UI grouping
- **`ShortcutHandler`** - Action handler with customization support
- **`ShortcutActionRegistration`** - Action handler binding

### Core Interface (`src/AIntern.Desktop/Services`)
- **`IKeyboardShortcutService`** - Enhanced interface with:
  - Context-aware dispatch
  - Action registration and lookup
  - Customization and persistence
  - Legacy compatibility with v0.3.5g API

### Service Implementation
- **`KeyboardShortcutService`** - 30+ default shortcuts for code generation workflow
- **`ShortcutManager`** - UI-level keyboard event handler with context detection

### Configuration
- **`AppSettings.CustomShortcuts`** - Persistence of shortcut customizations

## Tests

- 9 tests for `KeyboardShortcutService`
