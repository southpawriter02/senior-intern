# v0.5.5f - Terminal Settings Panel

**Released:** 2026-01-19

## Summary

Implements the Terminal Settings Panel UI with full settings management including appearance, behavior, and shell profile configuration. Features live preview of appearance changes, unsaved changes tracking, and reset-to-defaults functionality.

## Features

### TerminalSettingsViewModel
- Observable properties for all terminal settings (appearance, behavior, shell)
- Live preview via `PreviewSettingsChanged` event
- `HasUnsavedChanges` tracking with warning banner support
- Commands: `SaveSettings`, `ResetToDefaults`, `DiscardChanges`
- Shell profile management: `AddProfile`, `EditProfile`, `DeleteProfile`, `SetDefaultProfile`
- Validation with error display

### TerminalSettingsPanel.axaml
- **Appearance Section**: Font family (with preview), font size slider, line height, color theme, cursor style, cursor blink, bold-as-bright, ligatures toggle
- **Behavior Section**: Scrollback lines, bell enable/style, copy-on-select, scroll-on-input/output, workspace sync, confirm-on-close
- **Shell Profiles Section**: Profile list with icons, add/edit/delete buttons, default profile selector
- **Unsaved Changes Banner**: Warning with discard/save buttons

### ShellTypeToIconConverter
- Maps `ShellType` enum to appropriate icon geometries
- Supports: Bash, Zsh, PowerShell, Cmd, Fish, default
