# v0.4.5c Changelog - Partial Snippet Models

**Released**: 2026-01-16

## Summary

Data structures enabling partial code snippet application to specific file locations.

## Features

### Core Models (`src/AIntern.Core/Models`)

#### LineRange (Extended)
- Added `IsSingleLine` computed property
- Added `FromLine`, `EntireFile`, `FromZeroIndexed` factory methods
- Added `Contains(LineRange)`, `IsAdjacentTo` query methods
- Added `Merge`, `Intersect`, `Expand`, `Shift`, `ClampTo` transformations
- Added `ToZeroIndexedStart/End`, `EnumerateLines`, `ToDiffHeader` conversions
- Implements `IComparable<LineRange>`

#### New Types
- **`ColumnRange`** - 0-indexed column ranges (future character-level operations)
- **`SnippetInsertMode`** - Enum: Replace, InsertBefore, InsertAfter, Append, Prepend, ReplaceFile
- **`SnippetAnchorType`** - Enum for anchor matching (ExactText, Regex, FunctionSignature, etc.)
- **`SnippetAnchor`** - Text-based anchor with Pattern, Occurrence, Offset
- **`AnchorMatchResult`** - Result of anchor search with confidence
- **`IndentationStyle`** - Record for indentation detection (UseTabs, SpacesPerIndent)
- **`SnippetApplyOptions`** - Configuration record for snippet application
- **`SnippetContext`** - File context with line parsing and indentation detection

## Tests

- 11 new tests for `LineRange` extensions
- 6 new tests for `SnippetApplyOptions`
