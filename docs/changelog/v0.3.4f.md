# v0.3.4f: Editor Integration

**Released:** 2026-01-15

## Summary

Integrated context attachment functionality into the Editor Panel, enabling users
to attach code selections or entire files to chat via keyboard shortcuts.

## New Files

| File | Description |
|------|-------------|
| `SelectionInfo.cs` | Model for selection data |
| `SelectionAttachmentEventArgs.cs` | Event args for attachment |

## Modified Files

| File | Changes |
|------|---------|
| `EditorPanel.axaml.cs` | Attachment methods, event, keyboard shortcuts |
| `EditorPanelViewModel.cs` | HasSelection property |

---

## Key Features

| Feature | Description |
|---------|-------------|
| Ctrl+Shift+A | Attach current selection to chat |
| Ctrl+Shift+F | Attach entire file to chat |
| HasSelection | Enables/disables selection-based commands |
| AttachSelectionRequested | Event for integration |

## SelectionInfo Model

| Property | Type | Description |
|----------|------|-------------|
| FilePath | string | Full path to file |
| FileName | string | Display name |
| Language | string? | Detected language |
| Content | string | Selected text |
| StartLine | int | 1-based start line |
| EndLine | int | 1-based end line |
| IsFullFile | bool | Whether entire file |
