# Design Specification: The Senior Intern v0.3.2c "File Icon System"

## Executive Summary

This document provides a detailed implementation specification for v0.3.2c, which implements a comprehensive file icon system with SVG path data for 25+ file types, color brushes, and an Avalonia value converter for XAML bindings.

### v0.3.2c Scope

- Create `FileIconProvider` static class with icon paths and extension mappings
- Create `FileIconConverter` for XAML bindings
- Create `FileIcons.axaml` resource dictionary with colors and static geometries

### Key Deliverables

| Deliverable | Description |
|-------------|-------------|
| FileIconProvider | Static icon key/path provider |
| FileIconConverter | IValueConverter for PathIcon |
| FileIcons.axaml | Colors and static geometries |

```csharp
    public static string GetIconKeyForExtension(string extension)
    {
        if (string.IsNullOrEmpty(extension)) return "file";
        return ExtensionIconMap.TryGetValue(extension, out var key) ? key : "file-code";
    }

    public static string GetIconKeyForFileName(string fileName)
    {
        return fileName.ToLowerInvariant() switch
        {
            "dockerfile" => "file-config",
            "makefile" => "file-shell",
            "package.json" or "tsconfig.json" => "file-json",
            ".editorconfig" or ".prettierrc" => "file-config",
            "readme" or "readme.md" => "file-markdown",
            "license" or "license.md" => "file-text",
            _ => GetIconKeyForExtension(Path.GetExtension(fileName))
        };
    }

    public static string GetIconPath(string iconKey)
    {
        return IconPaths.TryGetValue(iconKey, out var path) ? path : IconPaths["file"];
    }

    public static IReadOnlyCollection<string> GetAllIconKeys() => IconPaths.Keys;
}
```

### Task 2: Create FileIconConverter

**File:** `src/SeniorIntern.Desktop/Converters/FileIconConverter.cs`

```csharp
namespace SeniorIntern.Desktop.Converters;

using Avalonia.Data.Converters;
using Avalonia.Media;
using SeniorIntern.Desktop.Services;
using System.Globalization;

/// <summary>Converts an icon key to a StreamGeometry for PathIcon.</summary>
public class FileIconConverter : IValueConverter
{
    public static FileIconConverter Instance { get; } = new();

    public object? Convert(object? value, Type targetType, object? parameter, CultureInfo culture)
    {
        var iconKey = value as string ?? "file";
        var pathData = FileIconProvider.GetIconPath(iconKey);
        return StreamGeometry.Parse(pathData);
    }

    public object? ConvertBack(object? value, Type targetType, object? parameter, CultureInfo culture)
        => throw new NotSupportedException();
}
```
