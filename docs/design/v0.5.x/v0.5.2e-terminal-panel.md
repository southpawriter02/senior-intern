# Design Specification: AIntern v0.5.2e "Terminal Panel"

## Overview

**Version**: v0.5.2e
**Parent**: v0.5.2 Terminal UI
**Focus**: Terminal panel view with tab bar, session tabs, and panel controls

### Purpose

This sub-version creates the terminal panel UI that integrates all previous v0.5.2 components:
1. `TerminalPanel.axaml` - Complete panel layout with tab bar and terminal content area
2. `TerminalPanel.axaml.cs` - Code-behind for session attachment and focus management

### Dependencies

**From v0.5.2a (Theme & Styles)**:
- Terminal color brushes: `TerminalBackground`, `TerminalTabBarBackground`
- Control themes: `TerminalTab`, `TerminalTabCloseButton`, `TerminalNewTabButton`, `TerminalIconButton`
- Icon paths: `TerminalIcon`, `PlusIcon`, `CloseIcon`, `MaximizeIcon`, `MinimizeIcon`, `ChevronDownIcon`

**From v0.5.2b (Terminal Renderer)**:
- `TerminalFontMetrics` for size calculations

**From v0.5.2c (Terminal Control)**:
- `TerminalControl` - Main input/rendering control with `AttachSessionAsync()`, `DetachSession()`, `Focus()`

**From v0.5.2d (ViewModels)**:
- `TerminalPanelViewModel` - Panel state, commands, session collection
- `TerminalSessionViewModel` - Session tab representation with `Id`, `Name`, `IsActive`
- `ActiveSessionChanged` event for view synchronization

**From Core Layer**:
- `ITerminalService` - Terminal session management

**Purpose**: Session attachment and focus management

```csharp
namespace AIntern.Desktop.Views;

using Avalonia.Controls;
using Avalonia.Interactivity;
using AIntern.Core.Interfaces;
using AIntern.Desktop.ViewModels;
using Microsoft.Extensions.Logging;

/// <summary>
/// Terminal panel view with tabs and session management.
/// Handles session attachment to the terminal control when the active session changes.
/// </summary>
/// <remarks>
/// Added in v0.5.2e.
/// </remarks>
public partial class TerminalPanel : UserControl
{
    private readonly ILogger<TerminalPanel>? _logger;
    private ITerminalService? _terminalService;
    private TerminalPanelViewModel? _viewModel;

    /// <summary>
    /// Creates a new TerminalPanel instance.
    /// </summary>
    public TerminalPanel()
    {
        InitializeComponent();
    }

    /// <summary>
    /// Creates a new TerminalPanel instance with logging.
    /// </summary>
    /// <param name="logger">Optional logger for diagnostics.</param>
    public TerminalPanel(ILogger<TerminalPanel>? logger) : this()
    {
        _logger = logger;
    }

    // â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•
    // Lifecycle
    // â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•

    protected override void OnLoaded(RoutedEventArgs e)
    {
        base.OnLoaded(e);

        _viewModel = DataContext as TerminalPanelViewModel;
        if (_viewModel != null)
        {
            _viewModel.ActiveSessionChanged += OnActiveSessionChanged;
            _logger?.LogDebug("Subscribed to ActiveSessionChanged");

            // Attach to current active session if one exists
            if (_viewModel.ActiveSession != null && _terminalService != null)
            {
                _ = AttachToSessionAsync(_viewModel.ActiveSession);
            }
        }
    }

    protected override void OnUnloaded(RoutedEventArgs e)
    {
        base.OnUnloaded(e);

        if (_viewModel != null)
        {
            _viewModel.ActiveSessionChanged -= OnActiveSessionChanged;
            _logger?.LogDebug("Unsubscribed from ActiveSessionChanged");
        }
    }

    // â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•
    // Public Methods
    // â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•

    /// <summary>
    /// Initializes the panel with the terminal service.
    /// Must be called by the parent window before sessions can be attached.
    /// </summary>
    /// <param name="terminalService">The terminal service for session management.</param>
    public void Initialize(ITerminalService terminalService)
    {
        _terminalService = terminalService ?? throw new ArgumentNullException(nameof(terminalService));
        _logger?.LogDebug("TerminalPanel initialized with terminal service");

        // If we already have a ViewModel with an active session, attach now
        if (_viewModel?.ActiveSession != null)
        {
            _ = AttachToSessionAsync(_viewModel.ActiveSession);
        }
    }

    // â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•
    // Event Handlers
    // â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•

    private async void OnActiveSessionChanged(object? sender, TerminalSessionViewModel? session)
    {
        if (_terminalService == null)
        {
            _logger?.LogWarning("Terminal service not initialized, cannot attach session");
            return;
        }

        if (session != null)
        {
            await AttachToSessionAsync(session);
        }
        else
        {
            DetachFromSession();
        }
    }

    // â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•
    // Private Methods
    // â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•â•

    private async Task AttachToSessionAsync(TerminalSessionViewModel session)
    {
        try
        {
            _logger?.LogDebug("Attaching to session {SessionId}", session.Id);
            await TerminalView.AttachSessionAsync(_terminalService!, session.Id);
            TerminalView.Focus();
            _logger?.LogDebug("Session {SessionId} attached and focused", session.Id);
        }
        catch (Exception ex)
        {
            _logger?.LogError(ex, "Failed to attach to session {SessionId}", session.Id);
        }
    }

    private void DetachFromSession()
    {
        _logger?.LogDebug("Detaching from current session");
        TerminalView.DetachSession();
    }
}
```
