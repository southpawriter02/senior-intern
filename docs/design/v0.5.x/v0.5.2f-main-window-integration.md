# Design Specification: AIntern v0.5.2f "Main Window Integration"

## Overview

**Version**: v0.5.2f
**Parent**: v0.5.2 Terminal UI
**Focus**: Integrate the terminal panel into the main window with resizable splitter and keyboard shortcuts

### Purpose

This sub-version completes the Terminal UI by integrating it into the application:
1. Update `MainWindow.axaml` to include the terminal panel with resize splitter
2. Update `MainWindow.axaml.cs` to initialize the terminal panel with the service
3. Update `MainWindowViewModel` to expose `TerminalPanelViewModel`
4. Add global keyboard shortcuts for terminal operations
5. Create `MultiplyConverter` for dynamic max height calculation
6. Update DI registration in `App.axaml.cs`

### Dependencies

**From v0.5.2d (ViewModels)**:
- `TerminalPanelViewModel` - Panel state and commands
  - `TogglePanelCommand`
  - `NewSessionCommand`
  - `CloseActiveSessionCommand`
  - `NextTabCommand`
  - `PreviousTabCommand`
  - `IsVisible`, `PanelHeight`, `IsMaximized`

**From v0.5.2e (Terminal Panel)**:
- `TerminalPanel` view with `Initialize(ITerminalService)` method

**From Core Layer**:
- `ITerminalService` - Terminal session management

**From Existing Desktop Layer**:
- `MainWindowViewModel` - Main window data context
- `MainWindow.axaml` - Existing main window layout

## File Specifications

### 1. MultiplyConverter.cs

**Location**: `src/AIntern.Desktop/Converters/MultiplyConverter.cs`
**Type**: `class` implementing `IValueConverter`
**Purpose**: Multiply a numeric value by a factor for dynamic height calculations

```csharp
namespace AIntern.Desktop.Converters;

using System.Globalization;
using Avalonia.Data.Converters;

/// <summary>
/// Multiplies a numeric value by a factor specified in the converter parameter.
/// Used for calculating dynamic max heights based on window dimensions.
/// </summary>
/// <remarks>
/// Added in v0.5.2f.
/// </remarks>
public class MultiplyConverter : IValueConverter
{
    /// <summary>
    /// Singleton instance for use in XAML resources.
    /// </summary>
    public static readonly MultiplyConverter Instance = new();

    /// <summary>
    /// Converts a numeric value by multiplying it with the parameter.
    /// </summary>
    /// <param name="value">The numeric value to multiply (typically a double).</param>
    /// <param name="targetType">The target type (not used).</param>
    /// <param name="parameter">The multiplier as a string (e.g., "0.7").</param>
    /// <param name="culture">The culture info (not used).</param>
    /// <returns>The multiplied value, or the original value if conversion fails.</returns>
    /// <example>
    /// Input: value=800.0, parameter="0.7"
    /// Output: 560.0
    /// </example>
    public object? Convert(object? value, Type targetType, object? parameter, CultureInfo culture)
    {
        if (value is double d && parameter is string s && double.TryParse(s, NumberStyles.Float, CultureInfo.InvariantCulture, out var factor))
        {
            return d * factor;
        }

        return value;
    }

    /// <summary>
    /// Not implemented - this is a one-way converter.
    /// </summary>
    public object? ConvertBack(object? value, Type targetType, object? parameter, CultureInfo culture)
    {
        throw new NotSupportedException("MultiplyConverter is a one-way converter.");
    }
}
```
